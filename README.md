# Tic Tac Toe

This project is a browser-based Tic Tac Toe game.

The goal of this project was to practice organizing JavaScript code, working with factory functions and modules, and building a complete interactive application from scratch.

Players take turns placing their marks on the board, and the game automatically detects wins, losses, and draws.

## Live Demo

https://alperen-isler.github.io/tic-tac-toe/

## Built with

- HTML
- CSS
- JavaScript

## Features

- Two-player gameplay
- choosable player Names
- Win and draw detection
- Dynamic game board updates
- Player turn tracking
- Player Score tracking
- Player color tracking
- Play another round functionality
- Responsive and clean interface
